Deploying the Products REST API

All the resources we have created in Octopus allow us to deploy our application to AWS, and the AWS resources we created provide the infrastructure required to expose those applications to end users. With this foundation in place it is now time to deploy our microservice stack.

The products microservice is the first to be deployed. This is a REST API exposing the books and articles that make up our online library. This service is written as a Lambda, and persists data in a serverless RDS database.

To create the AWS account in Octopus, apply the Terraform configuration in the terraform/products directory using the following commands: 

cd terraform/products
terraform init
terraform apply
